### How to contribute

* Clone repo
	```bash
	git clone git@github.com:Lipen/BioHack.git
	```

* Create a folder with your name
	```bash
	mkdir <your name>
	```

* Go to it
```bash
cd <your folder>
```

* Any time you want to share:
1. `git add <some file>` **OR** `git add -u .` **(NOTE DOT IN THE END)**
2. `git commit`
3. `git pull --rebase`
4. `git push`
